everything--文件搜索神器

10月 02, 2013 发布在 善用佳软

官方网站:www.voidtools.com

算是一款牛B到爆，如果你是小白第一次用类似的软件，绝对会产生一种想见恨晚的感觉。对比windows自带的搜索，everything几秒钟完成索引，瞬间搜索完成。再根据文件修改日期或者路径就能瞬间找到想要的文件。有人用了这个软件之后戏称平时热衷文件管理的人自从用了它文件都是随处乱放，反正根据文件名马上就能找到。呵呵，虽然是个笑话（个人认为有条理的整理文件还是很有道理的，比如要调出某个月和某个人的所有文件往来，随便放你就只能呵呵了），但是它给我们搜索文档带来的便利性可见一斑。办公族必备！

系统自带搜索和everything简单对比

everything:简单粗暴，实力见证一切，体积小，占用内存低，无广告不流氓，而且还是免费软件，太TM良心了~~

很多人说windows自带的搜索慢~~~其实这是一种错误的观念，君不见你每次用windows搜索它都会提示”在没有索引的位置搜索可能比较慢，请单击以添到索引..”
目测这句话被很多人无视了。其实创建索引后windows自身搜索也不是那么慢，只是创建更新索引的速度确实没那么让人满意。而且界面远远没有everything友好。。。。目测自带的搜索真是一无是处~~~是的，我也这么认为。但是就我看来自带的对比everything还是有2个很实在有用的功能：

  • 1.支持FAT32文件系统（everything只支持NTFS.但是在FAT32能无损转换成NTFS，且NTFS性能更优的情况下。。。。还有人会用FAT32么）>+<
  • 2.everything只支持文件名搜索。但是系统自带的还是有一定的OCR（Optical
    Character
    Recognition，光学字符识别）    能力。范例就是如果你有很多TIF扫描件，那么Windows TIFF
    IFilter    就对你很有用了。虽然搜索扫描件内容不是那么的精确，并不是每次都能找到。但是有总比没有强~~

另外everything还支持正则表达式（不会，不做评论）以及附带http和ftp服务器.

截图略过，开启和使用比较简单。经常用的是http服务器（ftp无搜索功能，且对中文字符乱码）。打开之后局域网内的电脑就能通过访问你的IP地址下载你电脑上的内容。比较蛋疼的是开启这个功能之后别人可以访问你电脑上的任何内容，加上everything快如闪电的搜索.呵呵，某个蛋疼的人用.mp4|.avi|.rmvb看到你的小秘密就不好咯,鉴于此对于局域网文件分享这个功能，Http
File Server更好用，如果你只用QQ发送文件…..当我没说~~~~

附送个不是技巧的技巧 比如你要搜索的文件在dropbox文件夹之下
，搜索的时候可以添加上 \dropbox

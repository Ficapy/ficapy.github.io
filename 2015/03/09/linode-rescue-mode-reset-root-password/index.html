Linode紧急模式修改密码

3月 09, 2015 发布在 随笔

VPS运行太久了，也没怎么上去看。前些天重装电脑没保存VPS的登陆信息。等需要上去看的时候才知道粗大事了！登陆不上去了

我前期因为安全需要设置了禁用密码登陆，改用密匙，修改了端口，无奈去问Linode客服，得到答复如下

继续问，我修改了登陆端口，而且希望修改密码机器不重启可以么，答曰没有办法。最后只能说这个官方复制的回复坑爹至极。
正确步骤如下：

  • 进入Remote Access页面获取Lisa via ssh代码
  • 使用SSH客户端连接，我使用的是cmder。该步骤要求你输入密码，密码为注册Linode的密码，非vps登陆密码
  • 进入后会出现如下字符localhost
    login:要求你输入账号密码，此时才是你的VPS登陆密码，非Linode账号密码（我就是在这里卡了好久。。。没反应过来）
  • 到这里就是上面官方客服的步骤了，1.先关闭VPS。2.进入rescue页面最下方重置root密码。3.开机
  • 用上面修改的root密码键入步骤三
  • 最后修改/etc/ssh/sshd_config配置service ssh restart
  • bingo，重置密码完成~~~~~~~~~·
